AIP Directory Structure

  • time/

    a link to a directory containing the times at which the AIP was constructed.

  • sha/

    a link to a directory containing each unique identifier for the index page of the AIP as it changes.

  • latest/

    a link to the latest built AIP.

  • latest.tar.gz

    a link to a download archive of the latest AIP.

  • nodate

    appears under several directories and contains the AIP with the date stripped from the filename. This allows permanent working links to the AIP documents.
